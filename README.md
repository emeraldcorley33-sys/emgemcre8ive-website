# emgemcre8ive website

Static marketing site for the emgemcre8ive design studio. Built as a single-page HTML/CSS experience with bold typography and a black/green palette.

## Stack
- HTML
- CSS

## Structure
- `index.html` — page markup with sections for hero, services, work, process, about, and contact.
- `style.css` — custom styling, layout, and responsive tweaks.

## Assets
Place the following next to the HTML/CSS or update the paths in `index.html`:
- `logo-emgemcre8ive.png`
- `hero-banner.png`

## Getting started
Open `index.html` in a browser. No build tools or dependencies are required.

## Notes
- Fonts: Space Grotesk and Manrope via Google Fonts.
- Palette: black/near-black backgrounds, white text, accent `#00bf63`.
